# Bigotti Online Store

Application e-commerce indépendante pour Bigotti Collection.

## Objectif

Permettre aux clients de consulter les articles Bigotti en ligne, choisir couleur et taille, passer commande, payer à la livraison ou via paiement carte simulé.

## Stack

- Frontend: Next.js + TypeScript + Tailwind CSS
- Backend: NestJS + Prisma
- Database: PostgreSQL
- Versioning: Git + GitHub

## Espaces

- Client
- Admin
- Manager
- Super Admin
